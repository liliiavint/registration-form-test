### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://github.com/liliiavint/48-registration-form-test.git
    ```
2. Install NPM packages
    ```sh
    npm i
    ```
    or
    ```sh
    npm install

3. Cypress - _install_
    ```
    npm install cypress --save-dev

4. Run the server
    ```sh
    npm run dev
    ```
